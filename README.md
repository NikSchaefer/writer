# Writer

A beautiful and focused text editor.

## 🚀 Quick Start

```bash
git clone https://github.com/NikSchaefer/writer

bun install

bun dev
```

## 🛠️ Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- shadcn/ui components

## 🎯 Features

- Clean, distraction-free writing environment
- Minimalist design
- Focus Mode to focus on writing
- Dark mode
- Local saving
- Fullscreen mode
